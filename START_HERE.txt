================================================================================
  🎯 WHAT'S DIFFERENT FOR EACH ROLE
================================================================================

SUPER ADMIN → Sees EVERYTHING (all features, full control)
TEACHER → Student/Class management, Attendance, Grades, Guidance, etc.
ADVISER → Same as Teacher but ONLY for their assigned section
GUIDANCE → Only Guidance Records (behavioral incidents)
NURSE → Only Clinic Visits (medical records)
REGISTRAR → Only Document Requests
EQUIPMENT ADMINS → Only their equipment category
FACILITIES → Only Facility Damage Records
